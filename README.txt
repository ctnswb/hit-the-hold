# Hit the Hold

> This is a Minimum Viable Product of an interactive climbing game based off of Randori Augmented Reality Climbing https://joinrandori.com/.

Built for a class project within 24 hours.  Not intended for commercial use.

## Usage

> You need two people to play this game. One will be the climber and one will be the controller. Use a projector to project the app onto a climbing wall.

## Requirements

- Node 6.10.2
